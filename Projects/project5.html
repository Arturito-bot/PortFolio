Système Embarqué pour Mini-Fusée

Implémentation d'un système embarqué de télémétrie pour l'acquisition et la transmission en temps réel des performances d'une fusée miniature.

Voir le code sur GitHub

Technologies & Matériel

C++ / Arduino Capteurs (MPU6050) Baromètre (BMP280) Module RF (LoRa) Conception PCB (KiCad) Soudure

Architecture du Système

1. Acquisition de Données (Vol)

Le cœur de la fusée est un microcontrôleur (ex: Arduino Nano) qui lit les données d'une centrale inertielle (MPU6050) pour l'accélération et l'orientation, et d'un baromètre (BMP280) pour l'altitude.

2. Transmission RF (Télémétrie)

Les données collectées sont formatées et envoyées en temps réel vers le sol via un module de communication longue portée (ex: LoRa ou XBee), permettant un suivi en direct de la trajectoire et de l'apogée.

3. Station au Sol & Visualisation

Une station de réception au sol (un autre module RF connecté à un PC) reçoit les paquets de données, les décode et les affiche sur une interface simple (ex: GUI en Python ou moniteur série) pour analyse post-vol.
